==================================
Management System - Action - Claim
==================================

This module link action and claim and is a dependency of:

* mgmtsystem_action
* mgmtsystem_claim
* crm_claim

**Table of contents**

.. contents::
   :local:

Installation
============

No installation required.

Configuration
=============

No configuration required.

Usage
=====

To use this module, you need to:

* go to Management Systems > Action -> Claim
* Link action to claim
